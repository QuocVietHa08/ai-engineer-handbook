# AI Engineer Handbook

Free curated preparation materials, roadmaps, projects, and interview guides for AI engineers.

This project follows the same broad architecture as Tech Interview Handbook: a Docusaurus documentation site with handbook content in `contents/`, sidebar navigation in `sidebars.js`, and a custom homepage in `src/pages/index.jsx`.

## Development

Install dependencies with your package manager, then start Docusaurus:

```bash
npm install
npm run start
```

The current environment has Node.js but does not have `npm`, `pnpm`, or `yarn` installed, so the site structure is scaffolded but not locally runnable here yet.
